What is Git?

Gitis a version control system; you can think of it as a series of snapshots (commits) of your code base. With the creation of each snapshot you build out a whole application, seeing the growth of a full app. During this you can make branches to experiment and come back to a previous snapshot you took or merge those experiments into the your app.

What is GitHub?

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere using those code snapshots that you took during development.

Git init

Git init is the act of registering this current code base.

Git add

Git add is how we stage the intergration of our current code.
